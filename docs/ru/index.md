## 🌟 Об OpenIPC

OpenIPC - это альтернативная [прошивка](https://github.com/OpenIPC) для IP-камер и своего рода зонтик для других проектов, которые являются компонентами нашей системы.

OpenIPC - это операционная система Linux на основе проектов Buildroot/OpenWrt, предназначенная для IP-камер с чипсетами от различных производителей, начиная с Goke GK72xx, HiSilicon Hi35xx, SigmaStar SSC33x и XiongmaiTech XM5xx.

Каждый желающий может использовать/вносить свой вклад в проект любым полезным способом!
