## Contributing

We welcome contributions to the OpenIPC project! See our [Contribution Guidelines](../reference/contribute.md) for information on how to get started.

### Ways to Contribute

1. **Code Contributions** - Submit pull requests on GitHub
2. **Documentation** - Improve documentation and guides
3. **Testing** - Test firmware on various hardware
4. **Bug Reports** - Report issues and suggest improvements
5. **Translation** - Help translate documentation
